# Як протекстувати систему
Пояснення надаються для ОС Windows.
1. Переконайтесь, що на локальній машині встановлений python 3.13 або новіший. Переконатись у цьому можна, якщо запустити у консолі (`Win+R` -> `cmd`) команду `python --version`. Сумісність зі старішими версіями python можлива, але не гарантується. За необхідності встановіть python з [офіційного сайту](https://www.python.org/).
2. Створіть рабочій каталог на вашій локальній машині, наприклад, `MAN_GA_Schedule\`.
3. Перенесіть вміст репозиторію в цю папку.
4. Відкрийте консоль (`Win+R` -> `cmd`)
5. Перейдіть в консолі в робочу папку (`cd $MAN_GA_Schedule$\MAN_GA_Schedule`)
6. Створіть віртуальне середовище командою
   ```
   python -m venv venv
   ```
7. Активуйте віртуальне середовище командою
   ```
   venv\Scripts\activate
   ```
8. Установіть необхідні модулі і залежності:
   ```
   pip install numpy openpyxl matplotlib deap
   ```
9. Тепер можна запускати обчислення
   ```
   python ga.py
   ```

Після запуску скрипту `ga.py` почнуться обчислення, які можуть тривати якийсь час у залежності від налаштувань у файлі `parameters.py`.
Після обчислень відкриється вікно з matplotlib-графіками збіжності еволюційного процесу.
Після закриття вікна відкриється 3D-візуалізація фронту Парето.
Після закриття вікна результати будуть збережені у папці `output/`

В даному репозиторії вже наявний налаштований файл `preferences.xlsx`. Якщо його потрібно переініціалізувати, це можна зробити запуском файлу `init.py`:
УВАГА! Ця процедура видалить наявний preferences.xlsx (якщо не змінити назву файлу у `config.py`)
```
python init.py
```
